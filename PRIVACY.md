# LazyLink Privacy Policy

Version: 1.0
Last updated: 2026-02-02

## Overview

LazyLink is a Chrome extension that allows users to save and organize links, articles, and YouTube videos directly in their browser.

## Data Collection

LazyLink does not collect any personal information or browsing history. The only data stored are:

* Links (URLs) saved by the user
* Titles of saved links
* Tags created by the user
* Pinned state of saved links

All data is stored locally in the browser using Chrome's storage.local API. No data is sent to any server or third party.

## How Data Is Used

* Saved links, titles, and tags are used to display and manage the user’s saved content in the extension.
* Data is never shared with anyone outside the user’s browser.

## Third-Party Sharing

LazyLink does not share user data with any third parties.

## Permissions Explanation

LazyLink requests the following Chrome permissions to function properly:

* activeTab – Access the current tab’s URL and title to save the page.
* tabs – Read the active tab’s URL and title.
* storage – Save links, titles, tags, and settings locally in the browser.
* contextMenus – Add right-click menu options to save links quickly.
* notifications – Show a notification when a link is successfully saved.
* scripting – Run scripts in the current tab to fetch page or link titles when saving links.
* host permissions (<all_urls>) – Access web pages to read link titles.

## Remote Code

LazyLink does not use remote code. All scripts run locally in the browser.
